🏥 TR Pharmacy — Modern React Pharmacy Web App








A modern React-based pharmacy website built for TR Pharmacy (Jankipuram, Lucknow).
The application provides an interactive and responsive platform where users can:

Browse medicines

Upload prescriptions

Order medicines

Contact pharmacy

View services and opening information

The project focuses on clean UI, smooth animations, and real-world pharmacy workflow simulation.

🌐 Project Overview

This project simulates a complete pharmacy website system including:

✔ Medicine catalog
✔ Cart system
✔ Prescription upload
✔ Contact form
✔ Services section
✔ Grand opening gallery
✔ Persistent storage using LocalStorage

✨ Key Features
💊 Medicine Catalog

Browse medicines by category

Search medicines instantly

View product descriptions

Upload medicine images

Add medicines to cart

🛒 Smart Cart System

Users can:

Add medicines to cart

Increase / decrease quantity

Remove items

Calculate total price automatically

Checkout with delivery details

Checkout flow:

1️⃣ Cart Items
2️⃣ Delivery Address
3️⃣ Payment Method

📋 Prescription Upload

Users can upload prescriptions by providing:

Patient name

Phone number

Delivery address

Prescription image

Uploaded prescriptions are previewed and stored locally.

⚙️ Installation
1️⃣ Clone Repository
git clone https://github.com/mo-arman/tr-pharmacy.git
2️⃣ Enter Project Folder
cd tr-pharmacy
3️⃣ Install Dependencies
npm install
4️⃣ Run Development Server
npm start

Open in browser:

http://localhost:3000
